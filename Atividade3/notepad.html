Notepad++

O Notepad++ foi desenvolvido para máquinas Windows e distribuído como software gratuito. Os usuários do Linux também podem usá-lo via Wine. Seu repositório também está disponível no GitHub. Como em outros projetos da comunidade, plugins de terceiros são suportados. O Notepad ++ diferencia-se pela sua simplicidade. Ele é super leve e existe até uma versão para celular, se você preferir.

Recursos

  • Notepad++ tem interface simples, leve e rápida.

  • Ele suporta o ambiente de codificação em várias linguages, do ActionScript, CSS, ao Visual Basic.

  • 100% compatível com Windows, embora outros OS não sejam suportados (sem software adicional).

Por que escolher?

  • É totalmente gratuito.

  • Extensível. Você pode adicionar plugins da comunidade ou criar seus próprios.

  • Customizável. Os desenvolvedores podem personalizar os recursos e as interfaces conforme suas preferências.
